# 16. Конструктор интерактивных панелей для выставок

- **Концепция**: No-code / Low-code платформа, позволяющая собрать для выставочного экрана перетаскиванием виджетов (графики, видеопотоки, интеграция с датчиками).
- **Инженерная новизна**: Плагинная архитектура фронтенда и системы динамического рендеринга UI на основе конфигурационных файлов (Schema-driven UI).
- **Архитектура и этапы реализации**:
  1. **Движок конфигурации**: Описание структуры экрана через JSON-схему.
  2. **Визуальный редактор**: Рабочее поле (Grid Layout) для размещения виджетов.
  3. **Модуль интеграции**: Механизм подписки виджетов на WebSockets для обновлений в реальном времени.
- **Стек технологий**: React / Vue.js, HTML5 Drag and Drop API, TailwindCSS, WebSockets.
- **Критерии успеха (KPI)**: Скорость рендеринга панели из 20+ виджетов < 300 мс; сохранение JSON-конфига без багов верстки.
